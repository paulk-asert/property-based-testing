# Example-based testing of a math method

## Instructions {.tabset}

Invoke the following `gradle` tasks:

### Windows

```
> gradlew SumBiggestPair:test SumBiggestPair:jacocoTestReport
```

### Linux/MacOS

```
$ ./gradlew SumBiggestPair:test SumBiggestPair:jacocoTestReport
```

## Result

You should expect to see the results in the `build/reports` directory.
